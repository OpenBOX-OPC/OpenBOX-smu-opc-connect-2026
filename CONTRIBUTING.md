# Contributing

Please submit registration and project information through the OpenBOX website.

Approved pull requests update the JSON files under `data/`. Keep the field names stable so the OpenBOX website can render the data automatically.

## Registration List

| Name | Introduction | Contact Method | Wants Team | Comment |
| --- | --- | --- | --- | --- |

Registration JSON fields:

| Field | Required | Description |
| --- | --- | --- |
| `githubUsername` | Yes | GitHub username from the logged-in account. |
| `name` | Yes | Participant full name or preferred display name. |
| `introduction` | Yes | Short personal introduction, skills, and experience. |
| `contactMethod` | Yes | Telegram, WeChat, email, or another reachable contact method. |
| `wantsTeam` | Yes | `yes`, `no`, or `maybe`. |
| `comment` | No | Extra notes. |
| `submittedAt` | Yes | ISO timestamp generated by the bot. |

## Project List

| Project | Track | Summary | Team | GitHub | Demo | Pitch Deck |
| --- | --- | --- | --- | --- | --- | --- |

Project JSON fields:

| Field | Required | Description |
| --- | --- | --- |
| `id` | Yes | URL-safe project id generated from the project name. |
| `name` | Yes | Project name. |
| `track` | Yes | Project category, such as `AI Agent`, `GameFi`, `Infra`, or `Others`. |
| `summary` | Yes | One short paragraph explaining what the project does. |
| `team` | Yes | Team or builder names. |
| `github` | No | GitHub repository URL. |
| `demo` | No | Demo URL. |
| `pitchDeck` | No | Pitch deck URL. |
| `members` | No | Array of member objects. |
| `submittedAt` | Yes | ISO timestamp generated by the bot. |

Pull requests are reviewed by OpenBOX maintainers before data is published.
